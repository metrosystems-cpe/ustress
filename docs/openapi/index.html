get /probe
Probes an endpoint (probe)
The configuration is set via query params

Query parameters

url (required)
Query Parameter — Endpoint you want to stress
requests (required)
Query Parameter — Number of hits format: int32
threads (required)
Query Parameter — Number of executors format: int32
method (required)
Query Parameter — Request method
insecure (optional)
Query Parameter — Hit insecure

Return type

AppResponse

get /reports

Query Parameter — Report id you want to fetch format: uuid
